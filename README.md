## Ongoing Development Agents

### Pippin

Fetch the targeted repo tree, process structure data, and generate standardized accessible URLs to enable AI models to parse, read and understand project resources efficiently.

### Muse

Arrange and customize multi-level nested JSON data structure, integrate descriptive notes & practical use cases for all specification JSON documents.

Implement full automatic Git workflow via subprocess, complete local commit, timestamp injection and remote push synchronization natively inside programs.

### Jimmy

Responsible for mutual format conversion between JSON files and Markdown documents, unify directory specification matching `--info--.md` rule.
